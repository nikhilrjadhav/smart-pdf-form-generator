# 📄 Smart PDF Form Generator

A lightweight web application that allows users to fill out a form, add a signature, and generate a downloadable PDF dynamically.

---

## 🚀 Features

- Interactive web form (HTML, CSS, JS)
- Signature capture using Canvas
- Dynamic PDF generation using TCPDF
- Each submission creates a new PDF (no overwrite)
- Clean folder structure (production-ready mindset)

---

## 🛠 Tech Stack

- PHP (Core)
- HTML, CSS, JavaScript
- TCPDF

---

## 📸 Demo Flow

1. Fill form
2. Draw signature
3. Click Generate PDF
4. Download new PDF instantly
